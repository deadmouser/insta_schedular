Connect Instagram
One-time setup — takes about 10 minutes
⚠️ You need a Creator or Business Instagram account. Personal accounts cannot use the API. Switching is free and reversible in IG Settings → Account → Switch to Professional Account.
Understanding Error 190 (Token Error)
If you are getting an Error 190, it means your token is invalid or expired. To fix this:
1. Go back to the Graph API Explorer.
2. Re-select your app and ensure instagram_basic and instagram_content_publish are checked.
3. Generate a new token and do not forget to exchange it for a 60-day long-lived token.
4. Paste the fresh token below.
Setup steps
1
Switch to Creator / Business on Instagram
IG App → Profile → ☰ → Settings → Account → Switch to Professional Account → pick Creator or Business.
2
Create a Meta Developer App
Go to developers.facebook.com/apps → Create App → Consumer → add Instagram Graph API product. Copy your App ID and App Secret.
3
Get a long-lived User Access Token
Use the Graph API Explorer → select your app → add permissions: instagram_basic instagram_content_publish pages_read_engagement → Generate Token → exchange for long-lived token (60 days).
4
Find your Instagram User ID
In Graph API Explorer run: GET /me/accounts → find your page ID → then GET /{page-id}?fields=instagram_business_account → copy the IG User ID (Not the Facebook Page ID!).
5
Paste credentials below & test
